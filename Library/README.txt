In order to successfully run this project you must add .env file to the root directory and set the following variables:

LIBRARY_DEPLOYMENT_ROPSTEN=< Ropsten deployment address >
PRIVATE_KEY__ROPSTEN=< Your private key ( Ropsten ) >
LIBRARY_DEPLOYMENT_LOCAL="0xeCc2da6c0217f340Eb7835a7e068730de962903F"
PRIVATE_KEY_LOCAL=< Your private key ( Ganache ) >
